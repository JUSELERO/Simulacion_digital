Vector:

Coleccin de uno o mas datos del mismo tipo.

Propiedades:
  • Tipo: Tiene el mismo tipo de datos siempre(Son atmicos).
  • Largo: Tamao del vector, cantidad de elementos, no tiene mas dimensiones.
  • Atributos: Pueden tener Metadatos de distintos tipos, describen al vector. 
    • ##Creamos un vector numrico

c(12,4,34,6,1,3)
    ## [1] 12  4 34  6  1  3
    ##asignamos el valor  un vector de strings
strings <- c("abc","dfg","hij","klm")

##imprimimos los strings
strings
    ## [1] "abc" "dfg" "hij" "klm"

    Agregar elementos a nuestro vector existente se puede de la siguiente manera: Asignando un valor, seguido del vector inicial:

    mi_vector <- c(TRUE, FALSE, TRUE, FALSE)

mi_vector <- c(mi_vector, FALSE)

#Ahora hay 5 elementos en el mismo vector, y se aadi un elemento booleane FALSE, al final del vector
mi_vector
    ## [1]  TRUE FALSE  TRUE FALSE FALSE
    #Se puede combinar vectores:
vector_1 <- c(1,2,3)
vector_2 <- c(4,5,6)
vector_3 <- c(vector_1,vector_2)
vector_3
    ## [1] 1 2 3 4 5 6
    #Podemos differenciar el tipo de elementos del vector 

class(vector_1)
    ## [1] "numeric"

    Pero se puede concatenar elementos de distinto tipo en un mismo vector, el resultado ser un vector con un mismo tipo, el mas cmodo que encuentre R.

    mi_vector_nuevo <- c(vector_1, "a")
mi_vector_nuevo# el nuevo vector sera de tipo string, pues es mas flexible para R.
    ## [1] "1" "2" "3" "a"

    Manipulacin de Vectores:

    Los vectores se pueden operar principalmente de forma:

    • aritmetica
    • racional 
    #Creamos un vector 

vector <- c(1,2,3,4,5)
vector +1
    ## [1] 2 3 4 5 6
    #Pero ah no ha cambiado el vector original, debemos asignarle la operacin aritmtica
vector=vector +1
vector
    ## [1] 2 3 4 5 6
    #Tambin podemos asignar nuevas operaciones

vector=vector *10# multiplicacin
vector
    ## [1] 20 30 40 50 60
    vector %% 6#residuo
    ## [1] 2 0 4 2 0
    ## Ahora la operacin racional:

vector > 7# compara cada elemento si es mayor que 7
    ## [1] TRUE TRUE TRUE TRUE TRUE
    vector == 30 # compara que valor es igual a 3
    ## [1] FALSE  TRUE FALSE FALSE FALSE

    ggplot(data = mtautos) +geom_point(mapping = aes(x = cilindrada, y = caballos))

    Ahora podremos agregarle una dimensin en colores, estos sern la cantidad de cambios que tienen en su caja de cambios.

    ggplot(data = mtautos) + geom_point(mapping = aes(x = cilindrada, y = caballos, color = cambios))

    ggplot(data = mtautos) + geom_point(mapping = aes(x = cilindrada, y = caballos)) + facet_wrap(~ cambios, nrow = 1)
